
# Employee Productivity Analysis (Excel Dashboard)

## Project Overview
This project analyzes employee productivity using Microsoft Excel. 
The dataset contains employee attendance, task completion, work hours, and productivity scores.

The analysis uses Pivot Tables and an interactive Excel dashboard to identify productivity trends across departments and job roles.

## Dataset Columns
- Name
- Department
- Job Role
- Attendance (%)
- Task Completion (%)
- Work Hours Logged
- Performance Score
- Productivity Score

## Key Metrics
- Average Productivity: 85.60
- Average Attendance: 87.46
- Average Task Completion: 84.97

## Analysis Performed
1. Average productivity by department
2. Average productivity by job role
3. Overall KPI metrics
4. Interactive dashboard visualization

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Excel Dashboard

## Dashboard
The dashboard shows productivity comparison across departments and job roles.
